# Introduction

Diffusive Labs REST API enables you to interact with various Language, Image, and Embedding Models using an API Key. Our API is designed to be OpenAI-compatible while providing additional features and optimizations.

## Authentication

All requests to the Diffusive Labs API must include an `Authorization` header with a valid API key.

### Headers Required
- `Authorization`: Bearer token with your API key
- `Content-Type`: application/json
